### 補足
これまで開発に関わってきたサービスは E コマース系が多めです。特段 E コマースが好きだからというわけではなく、過去積んできた経験が活きやすい領域がたまたま E コマースだったという成り行きです。現職はユアマイスター株式会社にて、サービス EC の YOURMYSTAR の機能開発やリニューアルを横断的に担当しています。

### 現職の業務内容
パブリッククラウド（AWS や Google Cloud 等）におけるインフラ設計・構築、コンテナ管理（ミドルウェア導入・設定含む）、CI/CD、アプリケーション開発（PHP 、Nuxt 等）を幅広くやっています（所謂フルスタックエンジニア）。現在の立場はテックリードに近い立ち回りをしておりまして、日々設計相談を受けたりもしています。また、過去2年ほどリーダーポジションでチーム開発をリードしていた時期もありました。

### Web エンジニアを志した経緯
大学では物理化学領域を専攻していましたが、同時にサークルのホームページを作ったりアフィリエイトを試したりするのを趣味レベルでやっていました。特にアフィリエイトを経験したことで「インターネットを介して1個人が収入源を得られる」ことにインターネットの可能性を大きく感じ、且つ「ホームページを作る楽しさの延長で手に職が得られたら良いなぁ」と思ったのが Web エンジニアを志そうと思ったきっかけでした。
